浏览器兼容的CSS Hack

兼容 IE 6,7,8,9 和 Firefox,Chrome 的 hack 写法

.demo{
	background-color:blue\9;  /* ie8 */
	*background-color:yellow; /* ie7 */
	_background-color:black;  /* ie6 */
}
:root .demo{
	background-color:green;   /* ie9,ff,chrome */
}

IE8 和 IE9 不需要区分的话还可以这样写

.demo{
	background-color:red;     /* ff,chrome */
	background-color:blue\9;  /* ie8,ie9 */
	*background-color:yellow; /* ie7 */
	_background-color:black;  /* ie6 */
}

清除浮动的方法

兼容各种浏览器的清除浮动方法

/* clear element float */
.cf:after {
	visibility: hidden;
	display: block;
	font-size: 0;
	content: " ";
	clear: both;
	height: 0;
}
.cf { *zoom: 1; } /* for IE6 IE7 */
