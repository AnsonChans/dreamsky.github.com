JavaScript 编写技巧

积累优秀的 JavaScript 编写方法。

回到页面顶部

jQuery实现回到页面顶部

$(window).scroll(function() {
	if ($(this).scrollTop() > 500) {
		$('#go-top').fadeIn();
	} else {
		$('#go-top').fadeOut();
	}
});
$('#go-top a').click(function() {
	$('body,html').animate({
		scrollTop : 0
	}, 500);
	return false;
});

jQuery prop 方法的使用

jQuery 1.6版本开始支持prop方法，可以在async,autofocus,checked,multiple,readonly,selected,disabled属性使用

$("#demo").prop("checked");
$("#demo").prop("disabled",true);

Bootstrap 导航切换

Bootstrap 导航动态切换的实现方法。

$(".nav-list > li").click(function(){
	if($(this).hasClass("help"))
		return;
	$(this).parent()
		.find(".active")
		.removeClass("active")
		.find("i").removeClass("icon-white");;
	$(this).addClass("active");
	$(this).find("i").addClass("icon-white");
	
	var tar=$($(this).find("a").attr("href"));
	$('body,html').animate({
		scrollTop: tar.offset().top-60
	}, "fast");
	return false;
});

禁止网页后退

奇特的禁止网页后退方法。

function changeHashOnLoad() {
     window.location.href += "#";
     setTimeout("changeHashAgain()", "50"); 
}

function changeHashAgain() {
  window.location.href += "1";
}

var storedHash = window.location.hash;
window.setInterval(function () {
    if (window.location.hash != storedHash) {
         window.location.hash = storedHash;
    }
}, 50);
<body onload="changeHashOnLoad();">
